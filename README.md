# 2. Contexto del Proyecto

**TravelHub - App Móvil** está desarrollada con React Native, y está disponible para dispositivos Android e iOS. Los viajeros "usuarios" pueden buscar hospedajes en su destino favorito, consultar el detalle del hospedaje y realizar reservas.

# 3. Instrucciones de Ejecución:

### Android:

**Nota:** La build de Android requiere Java 17. Asegurar exportar el JAVA_HOME antes de compilar Android.

    1. export JAVA_HOME=/Users/quantumsoft/Library/Java/JavaVirtualMachines/temurin-17.0.17/Contents/Home
    2. npx react-native run-android
### iOS:
    npx react-native run-ios
